\# RIPE Default Theme for GitBook

This is the default theme for the manuals for RIPE Atlas.

Published to npm as `gitbook-plugin-theme-ripe`

Can be used by specifying:

```sh
{...
'theme': 'theme-ripe',
'plugins': ['theme-ripe',...]
...}
```

in book.json

Has no configurable options at this time.
